# StreamFlix - Video Streaming Platform

A comprehensive video streaming platform built with React, demonstrating advanced media handling and user interface design for the JavaScript frameworks showcase website.

## Features

- **Adaptive Video Player**: Custom video controls with play/pause, seek, volume, and fullscreen
- **Playlist Management**: Interactive sidebar with video selection
- **Responsive Design**: Works seamlessly on all devices
- **Professional UI**: Netflix-style interface with smooth animations
- **Video Metadata**: Duration, quality, and view count display

## Technical Implementation

- **React Hooks**: Custom hooks for video player state and playlist management
- **Context API**: Global state management for video data
- **Component Architecture**: Modular, reusable video components
- **Event Handling**: Mouse and keyboard interactions for video controls
- **Responsive Grid**: Adaptive layout for different screen sizes

## Demo Data

- 5 sample videos with realistic metadata
- High-quality thumbnails from Picsum Photos
- Various video qualities and durations


## Integration

This project is designed to be embedded in the React framework showcase page as a live demonstration of React media handling capabilities.
