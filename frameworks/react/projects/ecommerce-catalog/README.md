# E-Commerce Catalog - React Demo

A comprehensive e-commerce product catalog built with React, demonstrating advanced frontend development skills for the JavaScript frameworks showcase website[1][2].

## Features

- **Infinite Scroll Product Catalog**: Loads products dynamically as user scrolls
- **Advanced Filtering**: Filter by category and sort by price/name
- **Real-time Search**: Instant product search functionality
- **Shopping Cart**: Add/remove products with cart management
- **Responsive Design**: Works seamlessly on all devices
- **Professional UI**: Matches the main website's design language

## Technical Implementation

- **React Components**: Modular component architecture
- **Context API**: Global state management for cart and filters
- **Custom Hooks**: Reusable logic for data fetching
- **Performance Optimization**: Throttled scroll events and efficient rendering
- **Modern CSS**: Professional styling with gradients and animations

## Demo Data

- 20 sample products across 6 categories
- High-quality images from Picsum Photos
- Realistic pricing and product information

## Integration

This project is designed to be embedded in the React framework showcase page as a live demonstration of React development skills.

## Technologies

- React 18 (via CDN)
- Babel for JSX transformation
- Modern CSS with CSS Grid and Flexbox
- ES6+ JavaScript features
