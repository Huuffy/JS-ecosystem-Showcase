## 🌟 About The Project


**JS ecosystem Showcase: The Best Use Cases** is not just a UI clone—it is a deeply technical, interactive portfolio built to answer the question: *"When should I use which JavaScript framework?"* 

There are countless ways to build a web application today. This project was born from my desire to dive deep into **11 different JavaScript frameworks/libraries** and create **55 targeted, real-world examples** that highlight exactly what makes each framework shine. Instead of building the same "To-Do List" 11 times, each framework section contains unique projects tailored to its specific architectural strengths.

### 🎯 Purpose
- **Demonstrate Versatility:** Showcase my ability to adapt to different component lifecycles, state management patterns, and build tools.
- **Architectural Understanding:** Prove that I understand *why* a framework should be chosen (e.g., using Svelte for compile-time performance vs. Angular for massive enterprise modules).
- **Consolidated Learning:** Act as a central, beautiful hub to access all of my framework-specific projects in one place.

## 🚀 Getting Started

To explore this repository locally and see the implemented frameworks and their use cases:

### Prerequisites

You need Node.js installed to run the Vite development server.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/viraj-bhatia/js-frameworks-portfolio.git
   ```
2. Navigate into the directory
   ```sh
   cd js-frameworks-portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
5. Click the `localhost` link provided in your terminal.
